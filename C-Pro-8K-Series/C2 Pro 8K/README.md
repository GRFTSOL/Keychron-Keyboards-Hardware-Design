# Keychron C2 Pro 8K — Hardware Design Files

> Documentation entry point for the C2 Pro 8K model directory in the C Pro 8K series.

## Current Status

This model directory is present in the repository, but design files have not been added yet.

## Expected File Types

When files are published for this model, they will typically appear in formats such as:

- STEP for enclosure and assembly models
- DWG / PDF for plate drawings
- STEP for stabilizer components
